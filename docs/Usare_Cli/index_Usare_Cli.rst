
**Usare la Command Line Interface (CLI)**
=========================================

La **Command Line Interface (CLI)** consente al personale tecnico di controllare e amministrare completamente ed ad alti 
livelli di granularità la Cloud Management Platform (**CMP**).
La **CMP** rappresenta l’interfaccia di comunicazione rispetto ai diversi back-end e permette l’automazione e l’orchestrazione 
dei servizi cloud veicolati all’utente finale tramite un portale web che ha l’obiettivo di semplificare l’esperienza utente 
nascondendo la complessità sottostante.
